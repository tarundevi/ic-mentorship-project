# S&P 500 Universe Expansion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expand the mean reversion strategy analysis from 5 stocks to the full S&P 500 universe, using batch download for speed, a pooled signal approach for statistical power, and cross-sectional distribution analysis to assess how consistently the strategies work.

**Architecture:** All new cells append after `output-summary` in `notebooks/main_analysis.ipynb`. A new `process_ticker_from_series` function replaces `run_pipeline`'s download step so batch-downloaded data can be fed in directly. All existing functions (`fit_garch`, strategy generators, `evaluate_signals`, etc.) are reused unchanged. Analysis runs in two modes: (1) pooled — all signals from all tickers concatenated, strategies evaluated as if one universe, (2) per-ticker — distribution of per-ticker mean returns to show consistency.

**Tech Stack:** Python 3, pandas, numpy, arch, yfinance (batch download), matplotlib, seaborn — no new packages needed. Runtime: ~20–30 min (GARCH is the bottleneck at ~2–3s/ticker).

```python
def process_ticker_from_series(ticker, adj_close_series):
    """Run full pipeline on a pre-downloaded adj_close Series. Returns processed DataFrame or None."""
    try:
        df = adj_close_series.dropna().to_frame('adj_close')
        df.index = pd.to_datetime(df.index)
        df = df.sort_index()
        df = df[~df.index.duplicated(keep='first')]

        if len(df) < max(100, WINDOW + 10):
            return None
        if df['adj_close'].std() == 0:
            return None

        df = compute_returns(df)
        if not np.isfinite(df['log_return']).any() or df['log_return'].std() == 0:
            return None

        df['cond_vol'] = fit_garch(df['log_return'])
        df['vol_regime'] = assign_volatility_regime(df['cond_vol'])
        df = compute_zscore(df)
        df = generate_signals(df)
        df = assign_signal_strength(df)
        df = compute_forward_returns(df)
        df['rsi'] = compute_rsi(df['adj_close'])
        df['ticker'] = ticker
        return df
    except Exception:
        return None
```

- Modify: `notebooks/main_analysis.ipynb` — append after Task 2 cell

- [ ] **Step 1: Add bulk runner cell**

Cell id: `sp500-bulk-runner`

```python
import time

sp500_dfs = {}
sp500_failed = []

total = len(sp500_tickers)
start_time = time.time()

for i, ticker in enumerate(sp500_tickers):
    if ticker not in sp500_adj_close.columns:
        sp500_failed.append(ticker)
        continue

    series = sp500_adj_close[ticker]
    result = process_ticker_from_series(ticker, series)

    if result is not None:
        sp500_dfs[ticker] = result
    else:
        sp500_failed.append(ticker)

    if (i + 1) % 50 == 0:
        elapsed = time.time() - start_time
        rate = (i + 1) / elapsed
        remaining = (total - i - 1) / rate
        print(f"  [{i+1}/{total}] {len(sp500_dfs)} ok, {len(sp500_failed)} failed — "
              f"{elapsed:.0f}s elapsed, ~{remaining:.0f}s remaining")

elapsed = time.time() - start_time
print(f"\nDone. {len(sp500_dfs)} tickers processed, {len(sp500_failed)} failed in {elapsed:.0f}s")
if sp500_failed:
    print(f"Failed: {sp500_failed[:20]}{'...' if len(sp500_failed) > 20 else ''}")
```

Expected: ~450–490 tickers successfully processed. Failures are typically tickers with insufficient history (recent additions to the index) or data gaps.

```python
per_ticker_rows = []

for ticker, df_t in sp500_dfs.items():
    row = {'ticker': ticker}
    for col, name in strategy_names.items():
        if col not in df_t.columns:
            df_t[col] = df_t['signal'] if col == 'sig_baseline' else \
                        generate_strong_only_signals(df_t) if col == 'sig_strong_only' else \
                        generate_strong_high_vol_signals(df_t) if col == 'sig_strong_high_vol' else \
                        generate_strong_rsi_confirmed_signals(df_t)
        active = df_t[df_t[col] != 0].dropna(subset=['fwd_return_5d'])
        if len(active) < 5:
            row[f'mean_5d_{name}'] = np.nan
            row[f'count_5d_{name}'] = 0
        else:
            strat_ret = active[col] * active['fwd_return_5d']
            row[f'mean_5d_{name}'] = strat_ret.mean()
            row[f'count_5d_{name}'] = int(strat_ret.count())
    per_ticker_rows.append(row)

per_ticker_df = pd.DataFrame(per_ticker_rows).set_index('ticker')
print(f"Per-ticker summary: {len(per_ticker_df)} tickers")
print("\nMedian mean_5d return by strategy:")
for name in strategy_names.values():
    col = f'mean_5d_{name}'
    vals = per_ticker_df[col].dropna()
    pct_positive = (vals > 0).mean() * 100
    print(f"  {name}: median={vals.median():.5f}, pct_positive={pct_positive:.1f}%")
```
